About the Machine

Firefox Tweet Machine is a visualization of Firefox activity on Twitter designed and put together for Mozilla by Quodis.

It is an experiment with exclusively open web tools and technologies, and works on all modern browsers that support HTML5 and CSS3.

Technologies used: Twitter API's (Search & REST); Box2DJS (ported to jQuery) for the physics; PHP, Simple PHP Proxy, Memcached, YAML loader and PHP Twitter OAuth for the backend; Raphäel JS for the gauge SVG animation; jQuery for easier javascript coding; ColorBox, history and timeago plugins and Topsy for the RT functions.
